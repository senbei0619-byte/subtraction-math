# 引き算 🧮

**短い説明:** シンプルなブラウザ上で動く「引き算」アプリです。ローカルで開いてすぐに使え、GitHub Pages で公開できます。

---

## ✨ 特長

- ブラウザだけで動作（`index.html`, `script.js`, `styles.css`）
- 軽量・オフラインでも利用可能
- 学習・デモ目的に最適

## 🚀 ローカルで使う方法

1. リポジトリをクローンまたはダウンロード
2. `index.html` をブラウザで開くだけ

## 📦 GitHub へ公開する手順（概要）

1. GitHub に新しいリポジトリを作成
2. このプロジェクトをプッシュ
3. GitHub のリポジトリ設定で **Pages** を有効化（ブランチ: `main` or `gh-pages`、パス: `/ (root)`）

詳細は `PUBLISH.md` を参照してください。
