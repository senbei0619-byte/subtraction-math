# Contributing 🤝

ご貢献ありがとうございます！簡単なガイドです。

## 参加方法
- Issue を作成して問題点を報告してください。
- 変更を提案する場合は、フォークしてプルリクエストを送ってください。

## プルリクエストの流れ
1. Issue を立てる（任意だが推奨）
2. ブランチを作る（例: `fix/typo`、`feature/new-demo`）
3. PR を送り、簡単な説明を付ける

## コードスタイル
- HTML/CSS/JavaScript の基本的な整形に従ってください（可読性が高いことが重要です）。

## テスト
- 小規模プロジェクトのため自動テストはありません。変更はブラウザで動作確認してください。

## ライセンス
- 送られた貢献物はプロジェクトのライセンス（`LICENSE`）に従うものとします。
